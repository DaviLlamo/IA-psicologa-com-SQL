# 🤖 IA Psicóloga com Memória

Projeto de API em Python usando FastAPI + Gemini para simular uma IA de apoio psicológico com armazenamento de histórico em MySQL.

## 🚀 Tecnologias
- FastAPI
- MySQL
- SQLAlchemy
- Gemini (Google GenAI)

## 📌 Funcionalidades
- Cadastro e login de usuários
- Armazenamento de conversas
- IA com resposta contextual
- Memória básica de histórico

## ▶️ Como rodar

```bash
pip install -r requirements.txt
uvicorn main:app --reload
